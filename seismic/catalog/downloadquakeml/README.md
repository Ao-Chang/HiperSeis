>

>
> python downloadquakeml.py --agency GA --start_time 2021-05-24T09:27:00 --end_time 2021-06-24T09:27:00 --min_magnitude 0 --max_magnitude 9.90 --bounding_box  [-58,100,-2,200] --output ~/data
